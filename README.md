<h1>How to run the app</h1>

<h2>Navigate to the backend folder</h2>
<p>Run the following command:</p>
<code>npm start</code> - <i>This allows us to run the Node.js server on port 5000</i>

<h2>Navigate to the frontend folder</h2>
<p>Run the following command:</p>
<code>npm start</code>
<br>
<p>The project will start on the localhost, on port 3000</p>
